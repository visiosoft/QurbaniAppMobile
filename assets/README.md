# Assets Folder

This folder should contain the following image assets for your Qurbani Mobile app:

## Required Assets

### 1. App Icon
- **File**: `icon.png`
- **Size**: 1024x1024 px
- **Description**: Main app icon that appears on the home screen

### 2. Splash Screen
- **File**: `splash.png`
- **Size**: 1242x2436 px (or larger)
- **Description**: Image displayed when the app is loading

### 3. Adaptive Icon (Android)
- **File**: `adaptive-icon.png`
- **Size**: 1024x1024 px
- **Description**: Android adaptive icon foreground image

### 4. Notification Icon
- **File**: `notification-icon.png`
- **Size**: 96x96 px (white on transparent)
- **Description**: Icon displayed in push notifications

### 5. Favicon (Web)
- **File**: `favicon.png`
- **Size**: 48x48 px
- **Description**: Favicon for web version

## Creating Placeholder Assets

If you don't have custom assets yet, you can create simple placeholder images:

1. Use a design tool like Figma, Canva, or even PowerPoint
2. Create images with the required dimensions
3. Use a simple design with your app name or a relevant icon
4. Save as PNG format

## Recommended Design

For a Qurbani management app, consider:
- Green color scheme (associated with Islam)
- Islamic geometric patterns
- Simple, clear iconography
- Crescent moon or star symbols
- Arabic calligraphy elements

## Quick Placeholders

For testing purposes, you can:
1. Create solid color images with text
2. Use online tools like https://placeholder.com
3. Use Expo's default assets temporarily

## Note

The app will work without these assets, but you'll see warnings in the console. For production, proper assets are required.
